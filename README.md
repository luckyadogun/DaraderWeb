# Michael_UBA_Rental_WEB

### Project setup
1. Create a virtual environment

- Install Python 3.7.3
`pip install pipenv`
`pipenv --python 3.7.3`

2. Activate virtual environment
`pipenv shell`

3. Install dependencies
`pipenv install -r requirements.txt`


### Database Setup

- Install Docker and Docker-compose

`docker-compose up`
